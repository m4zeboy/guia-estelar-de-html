Atributos

Servem como informações extras e configurações dos elementos.

Anatomia do Atributo

Nome do atributo, sinal de igual, dentro das aspas duplas o conteúdo do atributo.

src="imagem.jpg"

Exeistem atributos booleanos, ou sejam, não precisam de conteúdo, exemplo o atributo disabled do input.

Aspas

Omitir aspas até funciona, mas pode causar problemas dependendo do navegador, então, para evitar perder muito tempo caçando o erro no código, use sempre as aspas para delimitar o conteúdo do seu atributo.

Outra dica é usar aspas DUPLAS, pois quando o conteúdo do atributo conter alguma aspa o código ficará mais claro e entendível. ex: title="i don't know"
