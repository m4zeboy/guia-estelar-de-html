Aninhamento

Quando colocamos uma tag dentro de outra tag. Nesse caso, a palavra dentro está dentro da tag em, que por sua vez, está dentro de um parágrafo.

LEMBRE-SE ↓

Sempre que você aninhar uma tag dentro de outra, certifique-se de que ele foi fechada corretamente. (a tag filha deve ser fechada antes da tag mãe ser fechada, para evitar problemas e bugs futuros.)

Fluxo

Cada tag é lida de acordo com sua posição linear, ex: linha 1, linha 2...uma abaixo da outra.

Hierarquia

Tags dentro de outras devem seguir uma hierarquia, as tags dentro de uma são consideradas filha da tag que as envolve. Exemplo, um parágrafo dentro de uma tag div é filho da tag div.

Posicionamento dos elementos

Algumas tags ficaram na mesma linha no resultado final, mas outras poderão sofreer quebras de linha, dependendo de sua particularidade.

A tag p sofre essa quebra de linha sempre que adicionamos um novo parágrafo, mas quando quebramos a linha dentro do parágrafo com o enter, isso não será aplicado na visualização do documento.

A tag a e a tag em (hiperlinks) são alguns exemplos de tags que ficam na mesma linha, exemplo: link para lugar nenhum
