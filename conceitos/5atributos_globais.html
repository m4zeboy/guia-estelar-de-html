Atributos globais mais utilizados

Atributos globais são aplicados a todas as tags.

  • class: usado para classificar vários elementos pertencentes a uma mesma classe (css).
  • contenteditable: recebe true ou false, significa que você pode editar esse conteúdo.
  • data-*:: Posso colocar qualquer coisa depois do -, muito utilizado no Java script.
  • hidden: quando usado, esconde o elemento em que foi aplicado.
  • id: é o identificador único, só podemos aplicar um por elemento.
  • style: serve para aplicar estilizações dentro da própria tag.
  • tabindex: serve para a organizar a navegação usando a tecla tab.
  • title: Definir um título para o elemento.
